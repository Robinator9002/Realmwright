# Realmwright

Realmwright is a custom TTRPG engine and toolkit built as a learning project.

The long-term goal is to split the project into three clear layers:

- Rust for the engine, rules execution, validation, persistence, and core systems
- Python for tooling, scripting, simulations, and content workflows
- JavaScript for the user interface and interactive application layer

## Project Direction

Realmwright is intended to be homebrew-first rather than tied to one official ruleset.

The project is meant to help with:

- creating and managing player characters
- building custom classes, origins, items, spells, and abilities
- quickly drafting NPCs and enemies
- writing encounters and temporary stat blocks
- tracking inventory, effects, and campaign state

## Status

This repository is a fresh restart. The focus is on building the architecture cleanly from the beginning.
